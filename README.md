# User Management System

This is a Full Stack CRUD Application built as part of my Full-Stack Development Internship.

## Tech Stack
Backend:
- Node.js
- Express.js
- PostgreSQL

Frontend:
- HTML
- CSS
- JavaScript (Fetch API)

## Features
- Add new user
- View users
- Update user
- Delete user

## Project Structure
user-management-system
│
├── backend
├── frontend
└── database

## How to Run

1. Start PostgreSQL database
2. Run backend server
   cd backend
   node server.js

3. Open frontend
   Open index.html with Live Server
